Manuscript: What Layer 1 Cannot Measure
Layer 1 invariance testing is undefined when only a single deployment environment is available. SCI is identified by variation in feature statistics across environments; with a single environment there is no cross-environment variance to measure, and any computed value would be numerically meaningless — not zero, but zero by construction, since a single point has no variance. This is not a limitation of the v-REx approximation specifically: the IRM identification condition itself requires at least two environments with distinct data-generating processes to distinguish invariant from spurious features. When len(environments) < 2, the implementation returns assumption_violations=["single_environment: invariance untestable"] and value=None rather than a misleading scalar. Practitioners with access to only one site should instead use the subgroup disparity metrics in Layer 2, which can detect performance heterogeneity without requiring separate environments, at the cost of not being able to attribute that heterogeneity to spurious vs. invariant signal.
